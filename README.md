qn interactive map of Nepal that runs on web browser in html. also has node.js with tileserver.dl to load map offline as well if the api doesn't work

tips to use offline app:

1- install node.js and go to command prompt and use ""C:\nodejs\npm" install tileserver-gl" to install tileserver.gl inside npm folder.

2- run the server locally, with cmd "tileserver-gl" from the directory with .mbtiles

this will host the satellite map in your own server offline
